# Wemmies 💜

Wemmies is an Android application that provides a safe and anonymous space for users to share difficult thoughts ("Wemmies") and receive empathy from the community.

---

## Features

- 🔐 Google Sign-In with Firebase Authentication
- 💭 Create anonymous Wemmies
- 🌍 Community feed using Cloud Firestore
- ❤️ Send empathy to other users
- ✨ Wemmies transform after receiving 5 empathies
- 👤 User profile with live statistics
- 📸 Profile picture support
- 📊 Firebase Analytics

---

## Built With

- Java
- Android Studio
- Firebase Authentication
- Cloud Firestore
- Firebase Analytics
- RecyclerView
- Material Design
